# Projeto Drupal - Meu Projeto

## Descrição
Este é um projeto Drupal criado para o bootcamp de Drupal. Ele inclui funcionalidades para exibir e categorizar conteúdo dinâmico.

## Instruções para rodar
1. Clone o repositório.
2. Configure o ambiente local com Docker/DDEV.
3. Importe o banco de dados:
   
   docker cp [caminho_do_arquivo_database.sql] <nome_do_container>:/var/www/html/database.sql
   
4. Acesse o site no navegador.

## Dependências
- Docker
- DDEV
- Drupal

## Funcionalidades
- Exibição de conteúdos dinâmicos.
- Filtros personalizados por categoria.
